Simple personal clicker
launch, hold down W + S and a bunch of clicks of the left mouse button begin with an interval of 0.01 seconds, you can customize it for yourself

Простой личный кликер
запускаете, зажимаете W + S и начинаются куча кликов левоц кнопкой мыши промежутком 0.01 секунда, можно настроить под себя
